Python Tutorial - Tuples, Sets, and Dictionaries

Tuples in Python

A tuple is similar to a list, but it is immutable (unchangeable). Once created, you cannot modify its elements.

Characteristics:

  • Ordered
  • Unchangeable (immutable)
  • Allows duplicate elements

Creating Tuples

# Empty tuple
t1 = ()
print(t1)  # Output: ()

# Single-item tuple
t2 = ('hello',)
print(t2)  
print(type(t2))  # <class 'tuple'>

# Homogeneous tuple
t3 = (1, 2, 3, 4)

# Heterogeneous tuple
t4 = (1, 2.5, True, [1,2,3])

# Tuple with another tuple
t5 = (1, 2, 3, (4, 5))

# Tuple from a string
t6 = tuple('hello')
Output:
()
('hello',)
<class 'tuple'>
(1, 2, 3, 4)
(1, 2.5, True, [1, 2, 3])
(1, 2, 3, (4, 5))
('h', 'e', 'l', 'l', 'o')

Accessing Items

print(t3[0])      # 1
print(t3[-1])     # 4
print(t5[-1])  # 4

Editing Items

t3 = 100  # Raises TypeError

Operations

t1 = (1, 2, 3, 4)
t2 = (5, 6, 7, 8)

print(t1 + t2)  # Concatenation
print(t1 * 3)   # Repetition
print(1 in t1)  # Membership

Functions

t = (1, 2, 3, 4)
len(t)        # 4
sum(t)        # 10
min(t)        # 1
max(t)        # 4
sorted(t)     # [1, 2, 3, 4]

Difference between Lists & Tuples

  • Lists are mutable, tuples are immutable
  • Tuples are faster
  • Tuples use less memory
  • Lists have more built-in methods

Sets in Python

A set is an unordered collection of unique, immutable elements. The set itself is mutable.

Characteristics:

  • Unordered
  • Mutable
  • No duplicates allowed

Creating Sets

s = set()
s1 = {1, 2, 3}
s3 = {1, 'hello', 4.5, (1,2,3)} # Mixed data types
s4 = set([1,2,3])                # From list
s5 = {1,1,2,2,3,3}               # Duplicates removed

Set Operations

s1 = {1,2,3,4,5}
s2 = {4,5,6,7,8}

s1 | s2   # Union
s1 & s2   # Intersection
s1 - s2   # Difference
s1 ^ s2   # Symmetric difference

Frozenset

A frozenset is an immutable version of a set. It supports all read-only operations but no addition/removal. 

fs1 = frozenset([1,2,3])
fs2 = frozenset([3,4,5])
fs1 | fs2  # Union

Dictionaries in Python

A dictionary stores key-value pairs. Keys are unique and immutable.

Creating Dictionaries

# Empty dictionary
d = {}

# 1D dictionary
d1 = {'name':'nitish','gender':'male'}

# Mixed keys
d2 = {(1,2,3):1, 'hello':'world'}

# Nested dictionary
s = {
  'name':'nitish',
  'college':'bit',
  'subjects':{
     'dsa':50,
     'maths':67
  }
}

Accessing Items

my_dict = {'name': 'Jack', 'age': 26}
my_dict['age']
my_dict.get('age')
s['subjects']['maths']

Adding & Updating

d1['age'] = 30
s['subjects']['ds'] = 75

Removing Items

d1.pop('age')
d1.clear()
del s['subjects']['maths']

Dictionary Functions

len(d1)
sorted(d1)
d1.items()
d1.keys()
d1.values()
