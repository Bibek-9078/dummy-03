DAY-03: Python Lists Fundamentals

What is a list?

A list is a data type where you can store multiple items under one name. More technically, lists act like dynamic arrays, which means you can add more items on the fly.

Difference Between Lists and Arrays

ArraysLists
Arrays have a fixed size set during creation.Lists are dynamic and can change in size during runtime.
All elements in an array must be of the same data type.Lists can accommodate elements of different data types.
Memory for the entire array is allocated at once during initialization.Lists dynamically allocate memory as elements are added or removed.
Arrays provide constant-time access to elements through indexing.Lists may have slightly variable access time due to dynamic resizing.
Arrays are less flexible as their size cannot be easily changed.Lists are more flexible, allowing easy addition or removal of elements.
May lead to memory wastage if size is larger than necessary.More memory-efficient due to dynamic allocation.
Common in languages like C/C++.Native to Python and suited for most use-cases.

Array vs List (Memory Addresses)

L = [1, 2, 3]
print(id(L))     # Memory address of the list object L
print(id(L[0]))  # Memory address of integer 1 in the list
print(id(L[1]))  # Memory address of integer 2 in the list
print(id(L[2]))  # Memory address of integer 3 in the list
print(id(1))     # Memory address of integer 1
print(id(2))     # Memory address of integer 2
print(id(3))     # Memory address of integer 3

Creating a List

empty_list = []  # Empty List
print(empty_list)  # Output: []

homogeneous_list = [1, 2, 3, 4, 5]  # 1D Homogeneous List
print(homogeneous_list)  # Output: [1, 2, 3, 4, 5]

nested_list = [1, 2, 3, [4, 5]]  # 2D List (Nested List)
print(nested_list)  # Output: [1, 2, 3, [4, 5]]

three_d_list = [[[1, 2], [3, 4]], [[5, 6], [7, 8]]]  # 3D List
print(three_d_list)  # Output: [[[1, 2], [3, 4]], [[5, 6], [7, 8]]]

heterogeneous_list = [1, True, 5.6, 5+6j, 'Hello']  # Heterogeneous List
print(heterogeneous_list)  # Output: [1, True, 5.6, (5+6j), 'Hello']

string_list = list('hello')  # From string
print(string_list)  # Output: ['h', 'e', 'l', 'l', 'o']

Adding Items to a List

L = [1, 2, 3, 4, 5]
L.append(True)
print(L)  # Output: [1, 2, 3, 4, 5, True]

L = [1, 2, 3, 4, 5]

L = [1, 2, 3, 4, 5]
L.extend([6, 7, 8])
print(L)  # Output: [1, 2, 3, 4, 5, 6, 7, 8]

L = [1, 2, 3, 4, 5]
L.extend('delhi')
print(L)  # Output: [1, 2, 3, 4, 5, 'd', 'e', 'l', 'h', 'i']

L = [1, 2, 3, 4, 5]
L.insert(1, 100)
print(L)  # Output: [1, 100, 2, 3, 4, 5]

my_fruits = ['apple','kiwi','grapes','banana']

# (3,3) matrix (nested comprehension)
print([[i*j for i in range(1,4)] for j in range(1,4)])

# Cartesian products

print([i*j for i in L1 for j in L2])

Disadvantages of Python Lists

  • Slow (compared to arrays for large data)
  • Can consume more memory
  • Risky usage if not careful (mutable)

Common List Programs

# 1. Odd and even separation
L = [1, 2, 3, 4, 5, 6]
odd_numbers = [x for x in L if x % 2 != 0]
even_numbers = [x for x in L if x % 2 == 0]
print("Odd numbers:", odd_numbers)
print("Even numbers:", even_numbers)

# 2. Input list from user
input_str = input("Enter a list of numbers (e.g., [1, 2, 3]): ")
user_list = eval(input_str)
print("User list:", user_list)

# 3. Merge two lists without '+'
L1 = [1,2,3,4]
L2 = [5,6,7,8]
L1.extend(L2)
print(L1)

# 4. Replace all 3s with 300
L = [1, 2, 3, 4, 5, 3]
old_item = 3
new_item = 300
L = [new_item if x == old_item else x for x in L]
print("Updated list:", L)

# 5. Flatten a 2D list
def flatten_2d_list(lst):
    return [item for sublist in lst for item in sublist]
two_d_list = [[1, 2, 3], [4, 5], [6]]
one_d_list = flatten_2d_list(two_d_list)
print(one_d_list)

# 6. Remove duplicates from a list
def remove_duplicates(lst):
    seen = set()
    result = []
    for item in lst:
        if item not in seen:
            result.append(item)
            seen.add(item)
    return result
L = [1, 2, 1, 2, 3, 4, 5, 3, 4]
L_no_duplicates = remove_duplicates(L)
print(L_no_duplicates)

# 7. Check if a list is in ascending order
def is_ascending(lst):
    return all(lst[i] <= lst[i + 1] for i in range(len(lst) - 1))
L1 = [1, 2, 3, 4, 5]
L2 = [1, 3, 2, 4, 5]
print(is_ascending(L1))
print(is_ascending(L2))
