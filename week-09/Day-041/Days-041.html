Data Visualization Guide: From Data Types to Plots

This guide includes Python code with expected outputs for all plots.

2D Line Plot

Bivariate Analysis: Numerical vs Numerical
Use case: Visualizing trends over time

Basic Line Plot

price = [48000, 54000, 57000, 49000, 47000, 45000] year = [2015, 2016, 2017, 2018, 2019, 2020] plt.plot(year, price) plt.show()
<output: A line plot appears with year on the x-axis and price on the y-axis, showing the trend over time.>

The plot will display a line connecting the points (2015, 48000), ..., (2020, 45000).

Line Plot from Pandas DataFrame

batsman = pd.read_csv('/content/sharma-kohli.csv') plt.plot(batsman['index'], batsman['V Kohli']) plt.show()
<output: A line plot for Virat Kohli's runs scored over the index (e.g., seasons or matches).>

Multiple Line Plots

plt.plot(batsman['index'], batsman['V Kohli'], label='Virat') plt.plot(batsman['index'], batsman['RG Sharma'], label='Rohit') plt.legend() plt.show()
<output: Two lines, each representing runs scored by Virat Kohli and Rohit Sharma over the index. A legend identifies which line is which.>

Customizing Plots: Labels, Title, Colors, Line Styles

plt.plot(batsman['index'], batsman['V Kohli'], label='Virat', color='#D9F10F', linestyle='solid', linewidth=3, marker='D', markersize=10) plt.plot(batsman['index'], batsman['RG Sharma'], label='Rohit', color='#FC00D6', linestyle='dashdot', linewidth=2, marker='o') plt.title('Rohit Sharma Vs Virat Kohli Career Comparison') plt.xlabel('Season') plt.ylabel('Runs Scored') plt.legend(loc='upper right') plt.grid() plt.show()
<output: A professionally styled line plot with colors, line styles, markers, labels, title, legend, and grid. The plot clearly distinguishes between the two batsmen.>

Axes Limits, Grid, Saving

plt.ylim(0, 75000) plt.xlim(2017, 2019) plt.grid() plt.savefig('plot.png')
<output: The plot is restricted to the x-axis range 2017–2019 and y-axis range 0–75000. The plot is displayed with a grid and saved as "plot.png".>

Scatter Plots

Bivariate Analysis: Numerical vs Numerical
Use case: Finding correlation

Basic Scatter Plot

x = np.linspace(-10, 10, 50) y = 10*x + 3 + np.random.randint(0, 300, 50) plt.scatter(x, y) plt.show()
<output: 50 points scattered around an increasing line, showing a positive linear relationship between x and y.>

Scatter Plot from DataFrame

df = pd.read_csv('/content/batter.csv') plt.scatter(df['avg'], df['strike_rate'], color='red', marker='+') plt.title('Avg and SR analysis of Top 50 Batsmen') plt.xlabel('Average') plt.ylabel('Strike Rate') plt.show()
<output: Red plus-sign markers show each batsman's average (x) and strike rate (y). The plot helps visualize the relationship between the two.>

Marker Size, Color, and Alternatives

tips = sns.load_dataset('tips') plt.scatter(tips['total_bill'], tips['tip'], s=tips['size']*20) plt.show()
<output: Scatter plot of total_bill vs. tip, with marker size proportional to the party size.>
plt.plot(tips['total_bill'], tips['tip'], 'o') plt.show()
<output: Similar plot as above, but with plain circle markers and faster rendering.>

Bar Charts

Bivariate Analysis: Numerical vs Categorical
Use case: Aggregate analysis of groups

Simple Bar Chart

children = [10, 20, 40, 10, 30] colors = ['red', 'blue', 'green', 'yellow', 'pink'] plt.bar(colors, children, color='black') plt.xticks(rotation='vertical') plt.show()
<output: A black bar chart with five bars labeled by color. The categories are displayed vertically on the x-axis.>

Bar Chart from DataFrame (Multiple Columns)

df = pd.read_csv('/content/batsman_season_record.csv') plt.bar(np.arange(df.shape[0]) - 0.2, df['2015'], width=0.2, color='yellow') plt.bar(np.arange(df.shape[0]), df['2016'], width=0.2, color='red') plt.bar(np.arange(df.shape[0]) + 0.2, df['2017'], width=0.2, color='blue') plt.xticks(np.arange(df.shape[0]), df['batsman']) plt.show()
<output: Grouped bar chart with each group representing a batsman. Three bars per group (yellow for 2015, red for 2016, blue for 2017) show runs per season, side-by-side.>

Histograms

Univariate Analysis: Numerical Data
Use case: Frequency distribution

Basic Histogram

data = [32, 45, 56, 10, 15, 27, 61] plt.hist(data, bins=[10, 25, 40, 55, 70]) plt.show()
<output: Bar heights represent counts of data points in each bin (e.g., [25, 40): 2 bars).>
