What is Pandas?

Pandas is a fast, powerful, flexible and easy to use open source data analysis and manipulation tool, built on top of the Python programming language.

Official Documentation

Pandas Series

A Pandas Series is like a column in a table. It is a 1-D array holding data of any type.

Importing Pandas

import numpy as np
import pandas as pd
# No output, libraries imported successfully

Series from Lists

country = ['India','Pakistan','USA','Nepal','Srilanka']
pd.Series(country)
0 India 1 Pakistan 2 USA 3 Nepal 4 Srilanka dtype: object
runs = [13,24,56,78,100]
runs_ser = pd.Series(runs)
runs_ser
0 13 1 24 2 56 3 78 4 100 dtype: int64
marks = [67,57,89,100]
subjects = ['maths','english','science','hindi']
pd.Series(marks,index=subjects)
maths 67 english 57 science 89 hindi 100 dtype: int64

Series from Dict

marks = {
    'maths':67,
    'english':57,
    'science':89,
    'hindi':100
}
marks_series = pd.Series(marks,name='Nitish ke marks')
marks_series
maths 67 english 57 science 89 hindi 100 Name: Nitish ke marks, dtype: int64

Series Attributes

marks_series.size
4
marks_series.dtype
int64
marks_series.is_unique
True

Indexing

x = pd.Series([12,13,14,35,46,57,58,79,9])
x[-1]
9

Editing Series

marks_series['english'] = 100
marks_series
maths 67 english 100 science 89 hindi 100 Name: Nitish ke marks, dtype: int64

Handling Missing Values

temp = pd.Series([1,2,3,np.nan,5,6,np.nan,8,np.nan,10])
temp.isnull().sum()
3
temp.fillna(temp.mean())
0 1.0 1 2.0 2 3.0 3 5.0 4 5.0 5 6.0 6 5.0 7 8.0 8 5.0 9 10.0 dtype: float64
