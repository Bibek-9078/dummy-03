DAY-03: Python Strings Fundamentals

  • Strings are sequences of Characters.
  • In Python specifically, strings are a sequence of Unicode Characters.

1. Creating Strings

  • String can be created by enclosing characters inside single (' '), double (" "), or triple quotes (''' ''' or """ """).
  • Triple quotes are used to represent multiline strings.
# Single and double quotes
c = 'Hello'
print(c)
c = "Hello"
print(c)

# Multiline string using triple quotes
c = '''Hello
World'''
print(c)

s = str('hello')
print(s)
Hello Hello Hello World
hello

2. Accessing Substrings (Indexing and Slicing)

  • We can access individual characters using indexing and a range of characters using slicing.
  • The index starts with zeros.
  • Trying to access a character out of the index range will raise an error.
  • The index must be an integer; we cannot use float or other types, as this will result in a type error
  • Python allows negative indexes for its sequences
c = "hello"
print(c[0])          # Positive Indexing
print(c[-1])         # Negative Indexing

c = "Hello World"
print(c)
print(c[0:5])        # First 5 chars
print(c[2:])         # From index 2 to end
print(c[:4])         # From start to index 4
print(c[:])          # Full string
print(c[0:8:3])      # Slicing with step
print(c[0:6:-1])     # No output (wrong slicing)
h o Hello World Hello llo World Hell Hello World HlW

3. Editing and Deleting Strings

  • Python strings are immutable.
  • Strings are not deleted partially, but deleted completely.
  • Strings are not edited, but re-assigned.

6. Common String Functions

  1. len: It is a function that returns the number of items in an object.
  2. max: It is a function that returns the item with the highest value.
  3. min: It is a function that returns the item with the lowest value.
  4. sorted: It is a function that returns the sorted list of the specified iterable object.
c = "kolkata"
print(len(c))
print(max(c))
print(min(c))
print(sorted(c, reverse=True))
7 t a ['t', 'o', 'l', 'k', 'k', 'a', 'a']

7. String Methods

  1. capitalize: Converts the first character to uppercase.
  2. title: Converts the first character of each word to uppercase.
  3. upper: Converts a string into uppercase.
  4. lower: Converts a string into lowercase.
  5. swapcase: Swaps cases; lowercase becomes uppercase and vice versa.
  6. count: Returns the number of times a specified value occurs in a string.
  7. find: Searches the string for a specified value and returns the position of where it was found. Note: find() is used instead of index() to avoid ValueError.
  8. index: Searches the string for a specified value and returns the position of where it was found.
  9. endswith: Returns True if the string ends with the specified value.
  10. startswith: Returns True if the string starts with the specified value.
  11. format: Formats specified values in a string.
  12. isalnum: Returns True if all characters in the string are alphanumeric.
  13. isalpha: Returns True if all characters in the string are alphabetic.
  14. isdigit: Returns True if all characters in the string are digits.
  15. isidentifier: Returns True if the string is a valid identifier.
  16. split: Splits the string at the specified separator, and returns a list.
  17. join: Converts the elements of an iterable into a string.
  18. replace: Returns a string where a specified value is replaced with another specified value.
  19. strip: Returns a trimmed version of the string.

print("it is raining today".title())
print(c.upper())
print(c.lower())

Count frequency of a character in string

s = input('Enter the string:')
term = input('What would you like to search for:')
counter = 0
for i in s:
  if i == term:
    counter += 1
print('frequency', counter)
Enter the string: bibek What would you like to search for: b frequency 2

Remove a character from string

s = input('Enter the string:')
term = input('What would you like to remove:')
result = ''
for i in s:
  if i != term:
    result = result + i
print(result)
