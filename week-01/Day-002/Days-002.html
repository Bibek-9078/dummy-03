DAY-2 (OPERATORS IN PYTHON)

Operators are used to perform operations on variables and values. Python has the following operators:

  • 1. Arithmetic Operators
  • 2. Relational Operators
  • 3. Logical Operators
  • 4. Bitwise Operators
  • 5. Assignment Operators
  • 6.Membership Operators

1. Arithmetic Operators

Arithmetic operators are used to perform mathematical operations like addition, subtraction, multiplication, and division. There are 7 arithmetic operators in Python:

  • Addition(+):Adds two values.
  • Subtraction(-):Subtracts two values.
  • Multiplication(*):Product of two values.
  • Division: /Returns the quotient.
  • Modulus: %Returns the remainder.
  • Integer Division(//):Floor division.
  • Exponentiation:Raises the first operand to the power of the second.
a = 9
b = 2    # Define the variables
# Perform arithmetic operations and print results
print(a + b)  # Output: 11
print(a - b)  # Output: 7
print(a * b)  # Output: 18
print(a / b)  # Output: 4.5
print(a % b)  # Output: 1
print(a // b) # Output: 4
print(a ** b) # Output: 81

2. Comparison or Relational Operators

Comparison operators are used to compare two values:

  • Greater Than(>): If the value of the left operand is greater than the value of the right operand, then the condition becomes true.
  • Less Than(<): If the value of the left operand is less than the value of the right operand, then the condition becomes true.
  • Greater Than or Equal(>=): If the value of the left operand is greater than or equal to the value of the right operand, then the condition becomes true.
  • Less Than or Equal(<=): If the value of the left operand is less than or equal to the value of the right operand, then the condition becomes true.
  • Equal To(=): If the values of two operands are equal, then the condition becomes true.
  • Not Equal To(!=):If the values of two operands are Not equal, then the condition becomes true.
a = 9
b = 2        # Define the variables
# Perform comparison operations and print results
print(a > b)  # Output: True
print(a < b)  # Output: False
print(a >= b) # Output: True
print(a <= b) # Output: False
print(a == b) # Output: False
print(a != b) # Output: True

3. Logical Operators (AND, OR, NOT)

Logical operators are used to combine conditional statements.

  • OR: Returns True if one of the statements is true.
  • AND: Returns True if both statements are true.
  • NOT: Reverses the result.
a = True
b = False      # Define the Boolean variables
print(a or b)  # Output: True
print(a and b) # Output: False
print(not b)   # Output: True

4. Bitwise Operators

Bitwise operators are used to operate on binary numbers.

  • Bitwise AND (&): Sets each bit to 1 if both corresponding bits are 1.
  • Bitwise OR (|): Sets each bit to 1 if at least one of the corresponding bits is 1.
  • Bitwise XOR (^): Sets each bit to 1 if only one of the corresponding bits is 1.
  • Bitwise NOT (~): Inverts all the bits (0 becomes 1 and 1 becomes 0).
  • Bitwise Left Shift (<<): Shifts all bits to the left by a specified number of positions, filling with zeros from the right.
  • Bitwise Right Shift (>>): Shifts all bits to the right by a specified number of positions, filling with the leftmost bit.
X = 2  # Binary representation: 0010
Y = 3  # Binary representation: 0011
print(X & Y)   # Output: 2  (Binary: 0010 & 0011 = 0010) # Bitwise AND
print(X | Y)   # Output: 3  (Binary: 0010 | 0011 = 0011) # Bitwise OR
print(X ^ Y)   # Output: 1  (Binary: 0010 ^ 0011 = 0001) # Bitwise XOR
print(X >> Y)  # Output: 0  (Binary: 0010 >> 3 = 0000)   # Bitwise right shift
print(X << Y)  # Output: 16 (Binary: 0010 << 3 = 10000)  # Bitwise left shift
print(~X)      # Output: -3 (Binary: ~0010 = -0011 in two's complement) # Bitwise NOT
print(~Y)      # Output: -4 (Binary: ~0011 = -0100 in two's complement) # Bitwise NOT

5. Assignment Operators

Assignment operators are used to assign values to variables.

a = 2 # Define the variable
print(a)  # Output: 2
a += 3   # Increment a by 3
print(a)  # Output: 5
a -= 3   # Decrement a by 3
print(a)  # Output: 2

6. Identity Operators

Identity operators are used to compare the objects, not if they are equal, but if they are actually the same object, with the same memory location.

a = 3
b = 3   # Define the variables
# Check if a and b refer to the same object
print("a is b for integers 3 and 3:", a is b)  # Output: True

# Assign lists
a = [1, 2, 3, 4]
b = [1, 2, 3, 4]
print(a is b)  # Output: False

# Assign strings
a = "hello"
b = "hello"
print(a is b)  # Output: True

# Assign larger strings
a = "hello_world"
b = "hello_world"
print(a is b)  # Output: True

7. Membership Operators

  • in: Returns True if a sequence contains the specified value
  • not in: Returns True if a sequence does NOT contain the specified value
print('D' in 'Delhi')       # Output: True
print('D' not in 'Delhi')   # Output: False
print(1 in [2, 3, 4, 5, 6]) # Output: False
print(1 in [1, 2, 3])       # Output: True

8. if...elif...else Statement

  • The if...elif...else statement is used for decision making in Python.
  • The program evaluates the test expression and executes statement(s) only if the test expression is true.
  • If the test expression is false, the statement(s) are not executed.
email = input("Enter your email:")
password = input("Enter your password:")

if email == "bibek4panda@gmail.com" and password == "1234":
    print("welcome")
elif email == "bibek4panda@gmail.com" and password != "1234":
    print("password incorrect")
    password = input("Re-enter the password:")
    if password == "1234":
        print("finally correct")
    else:
        print("still incorrect")
else:
    print("incorrect")

10. Loops in Python

1. While Loop

Executes a block repeatedly as long as condition is True.

  • The while loop in Python is used to iterate over a block of code as long as the test expression (condition) is true.
  • The body of the loop is entered only if the test expression evaluates to true.
  • After one iteration, the test expression is checked again.
  • This process continues until the test expression evaluates to false.
number = int(input('enter the number:'))
i = 1
while i < 11:
    print(number, '*', i, '=', number * i)
    i += 1

While Loop with else

x = 1
while x < 3:
    print(x)
    x += 1
else:
    print('limit crossed')

9. Built-in Functions (Examples)

  • print(): Prints the specified message to the screen or other standard output device.
  • input(): Reads a line from the input (usually from the user) and converts it into a string.
  • type(): Returns the type of the object/data element.
  • int(): Converts the specified value into an integer number.
  • abs(): Returns the absolute value of a number.
  • pow(): Returns the value of x raised to the power of y.
  • min(): Returns the item with the lowest value, or the lowest value in an iterable.
  • max(): Returns the item with the highest value, or the highest value in an iterable.
  • round(): Returns a floating-point number that is a rounded version of the specified number.
  • divmod(): Returns the quotient and remainder when argument-1 is divided by argument-2.
  • bin(): Returns the binary representation of a number.
  • oct(): Converts a number into an octal value.
  • hex(): Converts a number into a hexadecimal value.
  • id(): Returns the ID (unique identifier) of an object.
  • ord(): Converts a character into its Unicode code point (integer).
  • len(): Returns the length of an object.
  • sum(): Returns the sum of items in an iterator.
